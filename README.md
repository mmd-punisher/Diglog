# Diglog

Aweblog project base on django

---

**View**

- Blog list
- Category filter
- Advance searching
    - Base on date, title, body and …
- Pagination

**User Blog**

- Sign up
    - Create acc
    - username and password
- Login
- Create a post
    - Title
    - Publish date
    - Status
    - Adding pics and ...
    - Add category
- Edit post
- Delete post
    - Warning message
- View counter
- Like and unlike a post, like counter
- Comment for a post
- Author info
- Update review
- …

**User Profile**

- Add picture to profile
- Edit profile
- …

**Admin Panel**

- Access to all blogs
- Access to users and permissions
- Filter base something
- Login admin
- …
